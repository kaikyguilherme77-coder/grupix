🔧 Como configurar pagamentos
💳
Mercado Pago (recomendado)
Aceita PIX, cartão e boleto. Taxa: 4,99%
1. Acesse developers.mercadopago.com
2. Crie um app → copie o Access Token
3. Adicione na Vercel: MP_ACCESS_TOKEN=seu_token
4. Configure webhook: grupix.site/api/webhook-mp
5. O arquivo api/create-payment.js já está configurado ✅
⚡
Fluxo de pagamento no site
Como o usuário paga
Usuário clica 🔥 Destacar → Escolhe plano → Paga no MP (PIX/cartão) → Grupo vai pro topo automaticamente ✅
📊 Projeção de receita
10 grupos pagando plano semanal (R$10):R$100/semana
20 grupos pagando plano mensal (R$25):R$500/mês
50 grupos pagando plano mensal (R$25):R$1.250/mês
AdSense (50k visitas/mês):+R$1.500/mês
